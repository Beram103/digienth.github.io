EXAMPLES FOR NUMBER SYSTEM

Binary digit 10101 = (1*2^4)+ (0*2^3)+(1*2^2)+(0*2^1)+(1*2^0) = 16 + 0 + 4 + 0 + 1
= 21 base 10

Binary

Decimal digit 2586 = (2*10^3)+(5*10^2)+(8*10^1)+ (6*10^0) = 2000 + 500 + 80 + 6

Decimal

3075 base 8 = (3*8^3)+(0*8^2)+ (7*8^1)+(5*8^0) =1536 + 0 + 56 + 5

Octal

8EF base 16 = (8*16^2)+(E * 16^1)+(F * 16^0) =8*256 + 14 * 16 + 15*1
=2048 + 224 + 15
= 431 base 10
